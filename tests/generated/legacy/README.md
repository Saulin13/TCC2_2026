# Testes gerados — layout antigo

Antes da padronização por dataset, os testes do TheAlgorithms ficavam em:

- `tests/generated/gpt/`
- `tests/generated/claude/`

O layout oficial atual é:

- `tests/generated/thealgorithms/gpt/`
- `tests/generated/thealgorithms/claude/`
- `tests/generated/real/gpt/`
- `tests/generated/real/claude/`

As pastas `gpt/` e `claude/` neste diretório `legacy/` são cópias do layout antigo, mantidas apenas como arquivo.
